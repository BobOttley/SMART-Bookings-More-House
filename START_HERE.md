# Start Here - Documentation Index

Welcome! This directory contains comprehensive documentation for the More House School Booking Platform.

## What is this platform?

The Booking Platform is a system that manages:
- School visits (Open Days, Private Tours, Taster Days)
- Family booking confirmations
- Tour guide assignments
- Automated email communications
- Feedback collection
- Conversion tracking (visit → enrollment)
- Analytics and reporting

## System Overview in 60 Seconds

**The system:**
- Allows families to book school visits online
- Lets admissions staff manage those bookings
- Automatically sends email reminders
- Collects feedback from families
- Tracks which families enroll

**Main features:**
- Parent booking form
- Booking management dashboard
- Tour guide assignments
- Email templates & automation
- Feedback forms
- Analytics dashboard

**Key tabs in admin dashboard:**
1. Dashboard - Manage bookings
2. Events & Tours - Create events
3. Tour Guides - Manage guides
4. Settings - Configure emails & timing
5. Analytics - View reports

**What admissions does daily:**
1. Check new bookings
2. Assign tour guides
3. Schedule confirmations
4. Check in families
5. Record outcomes
